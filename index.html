Approach for above Product Carousel

❯ The carousel is structured to include multiple items, each represented as a .carousel-item. Only one item is visible at a time.

❯ CSS transitions are used to create a smooth sliding effect.

❯ JavaScript is used to manage the current slide index and automatically rotate the carousel. The setInterval function is used for the automatic rotation, while manual navigation is enabled through arrow buttons.

❯ Responsiveness is ensured by using percentage-based widths and media queries to adjust text size on smaller screen.

Approach for above customizable Product

❯ The HTML structure consists of a selection interface with different chocolates, each having a name, price, and input field for the quantity. The data-price attribute is used to store the price of each chocolate.

❯ The CSS ensures that the layout is responsive and visually clean. Flexbox is used for a neat arrangement of chocolate items.

❯ The JavaScript logic handles the real-time calculation of total items and price. It ensures that the total number of chocolates does not exceed 8, and provides feedback to the user if they try to select more than 8 items.

❯ I chose vanilla JavaScript, HTML, and CSS to keep the solution lightweight and easily integrable into any project. This approach also makes it easy to understand and modify.
